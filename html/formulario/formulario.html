Formulário HTML

É uma seção de um documento que contém controles como campos de texto,
campos de senha, caixas de seleção, botões de opção, botão de envio, menus etc.

Um formulário HTML facilita o usuário a inserir dados que devem ser
enviados ao servidor para processamento, como nome, endereço de e-mail,
senha, número de telefone, etc.

Por que usar o form HTML

Formulários HTML são necessários se você quiser coletar alguns dados do visitante do site.

Por exemplo: Se um usuário quiser comprar alguns itens na internet, ele / ela deve preencher o formulário, como endereço de entrega e detalhes do cartão de crédito / débito para que o item possa ser enviado para o endereço fornecido.

Sintaxe de formulário HTML

<form action="server url" method="get|post">

O elemento HTml <form>

O elemento fornece uma seção de documento para receber a entrada do
usuário. Ele fornece vários controles interativos para enviar
informações para o servidor web, como campo de texto,
área de texto, campo de senha, etc.

Nota: O elemento <form> não cria um formulário em si,
mas é um contêiner para conter todos os elementos de formulário necessários,
como <input> <label>, etc.

O elemento Html <input>

O elemento <input> é fundamental do formulário. Ele é usado para criar
campos de formulário, para receber informações do usuário. Pode-se aplicar diferentes
entradas arquivadas para coletar diferentes informações do usuário do formulário

O elemento input é o elemento de formulário mais usado.
Ele pode ser exibido de muitas maneiras, tudo depende do atributo que recebe.
Se type="text"> Exibe um campo de entrada de texto de linha única
Se type="rádio"> Exibe um botão de opção (para selecionar uma das muitas opções)
Se type="caixa de seleção"> Exibe uma caixa de seleção (para selecionar zero ou mais de muitas opções)
Se type="submit"> Exibe um botão de envio (para enviar o formulário)
Se type="button"> Exibe um botão clicável.
E muito mais.

Exemplo: <input type="radio">
